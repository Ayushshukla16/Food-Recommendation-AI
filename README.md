# 🍔 Food Recommendation System (Tkinter UI)

A modern **Food Recommendation System** built using **Python and Tkinter** with an animated UI.
This application suggests food based on **diet, meal type, and mood**, with a beautiful dark theme and moving background effects.

---

## 🚀 Features

* 🎯 Smart food recommendations based on:

  * Diet (Veg / Non-Veg)
  * Meal (Breakfast / Lunch / Dinner)
  * Mood (Happy / Sad / Stressed / Lazy)

* 🎨 Modern UI:

  * Dark theme design
  * Animated background (floating particles 🌊)
  * Clean and responsive layout

* ✨ Interactive Experience:

  * Typing animation for results
  * “Thinking...” delay (AI-like feel)
  * Randomized suggestions for variety

* 🍽️ Rich Food Database:

  * Indian dishes 🇮🇳
  * Fast food 🍕
  * Snacks & comfort food 🍟
  * Healthy options 🥗

---

## ▶️ How It Works

1. User selects:

   * Diet
   * Meal type
   * Mood

2. System processes input using:

   * Rule-based filtering
   * Weighted random selection

3. Displays a **personalized food recommendation** 🎉

---

## 🧠 Recommendation Logic

* Gives priority to:

  * Selected diet + meal combination
* Adds:

  * Mood-based suggestions
* Includes:

  * Random variety for better experience
